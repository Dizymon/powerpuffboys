# Web Medical Appointment System using CRUD Operations and Interval Searching

A web-based platform for booking medical appointments, managing patient records, and scheduling for doctors. Patients can book appointments with licensed doctors, while doctors can manage schedules, approve requests, and update professional profiles.

---

## Features

- **User Accounts:** Secure login with role-based access (Patient / Doctor)  
- **Appointment Scheduling:** Book, reschedule, and cancel appointments  
- **Approval Workflow:** Doctors can approve or reject requests; patients track status  
- **Medical Records:** Store and manage patient info  
- **CRUD Operations:** Doctors manage available appointment slots  

---

## User Roles

- **Patient:** Browse doctors, book appointments, track approval status  
- **Doctor:** Approve/reject requests, manage availability, update profile  

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask)  
- **Database:** SQL (MySQL/PostgreSQL)  
- **Architecture:** RESTful API, Role-based routing
